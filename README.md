  项目背景：
===
  自从接触vue以来，一直对它很感兴趣。最近使用vue2.0写了个外卖商家页面来练练手，目前这个版本还很简单且粗糙，日后待我慢慢修改完善。

  技术栈：
---
 * vue2.0
 * vue-router
 * axios
 * vue-cli
 * webpack
 * ES6
 * less
 * better-scroll
 * moment.js
 * progressbar.js
 * swiper.js
 * zepto.js
 * fonts
 * flex等
